09 - Listing & Public Web Traces

5707 Scenic Ave - Listing & Public Web Traces

Why this note exists

Public real-estate aggregation data around this property is messy. This file tracks what appears to be signal vs noise.

Public web traces found so far

Homes.com sold-page snippet

A Brave result for Homes.com surfaced a useful snippet indicating:

  • 5707 Scenic Ave, Mexico, NY 13114
  • Sold Sep 03, 2024
  • 2,184 sq ft
  • 54 days on market
  • asking/recorded sale context aligning with the county transaction date
  • listing flavor text describing an updated grand brick home in the Village of Mexico

This is directionally consistent with the county record and helps support a real 2024 market listing cycle.

Public current-listing trace

A Century 21 Galloway search result snippet shows:

  • 5707 Scenic Avenue appears in current Mexico inventory
  • price seen in snippet context: $339,900

This should be treated as a current-market clue, not yet a final verified list-price record until pulled directly from the listing page or MLS-backed source.

Data-quality problems encountered

Aggregator mismatch example

One Homes.com snippet also mixed in obviously incorrect attributes like:

  • built 1989
  • 4 beds / 3 baths
  • 2,027 sq ft
  • sold Aug 30, 2024 / 4 DOM

That appears to be a search-index contamination / adjacent listing bleed, not trustworthy property data.

Practical conclusion

Use public listing aggregators for:

  • rough confirmation that the property was listed and sold in 2024
  • rough hints about current asking price
  • photo discovery leads

Do not use them as final truth for:

  • bed/bath count
  • year built
  • DOM
  • property type
  • current configuration

Best next steps

  • pull current listing page directly from brokerage / MLS-backed source
  • capture listing photos and remarks into folder
  • capture 2024 sale listing details if older cached page or history is available
  • compare listing remarks for whether it was marketed as:
  • single family
  • two family
  • in-law / accessory setup
  • historic home
