# house-search

Research and presentation repo for historic-house/property investigations, with the current primary focus on **5707 Scenic Ave, Mexico, NY**.

The goal is to turn scattered notes, source captures, photos, public records, listings, historic references, and field observations into a structured, reviewable web archive.

## Current focus: `5707/`

The `5707/` folder is the canonical web package for the Scenic Avenue house research. It includes:
- page-based research notes
- media archives
- source extracts
- listing recovery artifacts
- field-capture imagery
- genealogy / family-history research connected to the house
